## Responsive Image Naming Convention

Images with responsive variants follow this naming pattern:
- `{name}.webp` - Original/default size
- `{name}-640w.webp` - 640px width (mobile)
- `{name}-828w.webp` - 828px width (tablet small)
- `{name}-1080w.webp` - 1080px width (tablet large)
- `{name}-1200w.webp` - 1200px width (desktop)
- `{name}-1920w.webp` - 1920px width (large displays)

Most images are available in both `.jpg` and `.webp` formats for browser compatibility.
