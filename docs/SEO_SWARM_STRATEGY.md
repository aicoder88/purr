**Best Practices:**
- Front-load primary keyword within first 35 characters
- Include brand name at end (separated by `|`)
- Use power words: "Best", "Eliminate", "Natural", "Proven"
- Add urgency/modifiers: "2026", "Guide", "Reviews"

**Character Count Guidelines:**
- English: 50-60 chars
- French: 50-60 chars (accents count as 1)
- Spanish: 50-60 chars

## 4. Technical SEO Requirements

### 4.1 Hreflang Tag Implementation

**Implementation Pattern:**
```html
<!-- In <head> section -->
<link rel="alternate" hreflang="en-ca" href="https://www.purrify.ca/page" />
<link rel="alternate" hreflang="fr-ca" href="https://www.purrify.ca/fr/page" />
<link rel="alternate" hreflang="zh-cn" href="https://www.purrify.ca/zh/page" />
<link rel="alternate" hreflang="es" href="https://www.purrify.ca/es/page" />
<link rel="alternate" hreflang="x-default" href="https://www.purrify.ca/page" />
```

**Next.js Implementation (app router):**
```typescript
// app/[locale]/page.tsx or layout.tsx
export async function generateMetadata({ params }: Props): Promise<Metadata> {
  const { locale } = await params;
  const baseUrl = 'https://www.purrify.ca';
  const path = '/products'; // current page path
  
  return {
    alternates: {
      canonical: `${baseUrl}${locale === 'en' ? '' : `/${locale}`}${path}`,
      languages: {
        'en-CA': `${baseUrl}${path}`,
        'fr-CA': `${baseUrl}/fr${path}`,
        'zh-CN': `${baseUrl}/zh${path}`,
        'es': `${baseUrl}/es${path}`,
        'x-default': `${baseUrl}${path}`,
      },
    },
  };
}
```

**Validation Checklist:**
- [ ] Every page has self-referencing hreflang
- [ ] All language variants link to each other (bidirectional)
- [ ] x-default points to English version
- [ ] No missing return links
- [ ] URLs are absolute (https://...)
