## Implementation Pattern Comparison

### Pattern 1: `useEnhancedSEO` + `NextSeo` (Recommended ✅)

**Used by:** `/try-free`, `/results`

```typescript
const { nextSeoProps, schema } = useEnhancedSEO({
  path: '/page',
  title: 'Page Title',
  description: 'Page description',
  targetKeyword: 'primary keyword',
  schemaType: 'product', // or 'article', 'faq', etc.
  schemaData: { ... },
  keywords: ['keyword1', 'keyword2'],
});

return (
  <>
    <NextSeo {...nextSeoProps} />
    {schema && <script type="application/ld+json" dangerouslySetInnerHTML={generateJSONLD(schema)} />}
    {/* page content */}
  </>
);
```

**Benefits:**
- Full feature set (OG, Twitter, alternates, canonical)
- Automatic optimization via `optimizeMetaTitle()` and `optimizeMetaDescription()`
- Schema support with multiple types
- Clean, declarative code

## Recommendations Summary

### Critical Fixes Required

| Priority | Page | Issue | Fix |
|----------|------|-------|-----|
| High | `/terms` | OG/Twitter tags missing, description short | Replace manual meta with `<NextSeo />`, expand description |
| High | `/documents` | OG/Twitter tags missing, description short | Replace manual meta with `<NextSeo />`, expand description |
| Medium | `/try-free` | Title 66 chars (6 chars over) | Shorten title to 60 chars max |

### Best Practices for Client-Side SEO

1. **Always use `<NextSeo {...nextSeoProps} />`** instead of manual meta tag rendering to ensure all SEO features are applied.

2. **Title Length:** Keep between 50-60 characters
   - Optimal: 55-58 characters
   - Google typically displays first 50-60 characters

3. **Description Length:** Keep between 140-155 characters
   - Optimal: 150-155 characters
   - Google typically displays up to 160 characters

4. **Always specify `targetKeyword`** in useEnhancedSEO for automatic optimization

5. **Use appropriate schema types:**
   - Product pages → `schemaType: 'product'`
   - Blog posts → `schemaType: 'article'`
   - FAQ pages → `schemaType: 'faq'`
   - Location pages → `schemaType: 'location'`

6. **Include keywords array** for meta keywords tag (still used by some search engines)

7. **Leverage built-in features:**
   - Language alternates (auto-generated)
   - Canonical URLs (auto-generated)
   - OG image generation
   - Twitter card configuration

### Suggested Implementation Template

```typescript
'use client';

import { useEnhancedSEO } from '@/hooks/useEnhancedSEO';
import { generateJSONLD } from '@/lib/seo-utils';

export default function PageName() {
  const { nextSeoProps, schema } = useEnhancedSEO({
    path: '/page-path',
    title: 'Optimal Title Here (50-60 chars)',
    description: 'Compelling description here (140-155 chars) that includes target keywords naturally.',
    targetKeyword: 'primary keyword',
    schemaType: 'product', // or 'article', 'faq', etc.
    schemaData: {
      // Schema-specific data
    },
    keywords: ['keyword1', 'keyword2', 'keyword3'],
  });

  return (
    <>
      <NextSeo {...nextSeoProps} />
      {schema && (
        <script
          type="application/ld+json"
          dangerouslySetInnerHTML={generateJSONLD(schema)}
        />
      )}
      {/* Page content */}
    </>
  );
}
```
