## Image Categories Summary

### Foreground Images
**Definition**: Images that are primary content, visible to users as distinct elements

**Common Types**:
- Product photos (20g.webp, 60g.webp, 140g.webp)
- Hero images (cat photos, lifestyle shots)
- Step-by-step instruction images
- Scientific diagrams and illustrations
- Logo (purrify-logo.png)
- Unsplash stock photos (blog posts)
- Generated images (learn pages)

**Typical Count Per Page**: 2-6 images

### Background Images
**Definition**: Images used as decorative backgrounds, often with overlays or reduced opacity

**Common Types**:
- Gradient mesh backgrounds (CSS-based)
- Decorative blur elements
- Hero section backgrounds
- Section divider backgrounds

**Typical Count Per Page**: 1-2 images

### Charts & Graphics
**Definition**: Data visualizations, comparison tables, and infographics

**Common Types**:
- Comparison tables (products, solutions)
- Process flow diagrams
- Scientific diagrams
- Results/data charts
- Timeline graphics

**Typical Count Per Page**: 0-3 charts

**When updating "all pages on the website":**

   - All 13+ Location pages (SEO)

**Image Optimization Requirements:**
- All foreground images should have proper `alt` text
- Background images should use `aria-hidden="true"` or be CSS-only
- Charts should have accessible table alternatives
- Product images must be optimized (WebP/AVIF)
